Pour Chrome :
Axi est maintenant disponible depuis le chrome store ! Télécharge là avec le lien ci-dessous :
Axi pour Chrome !

https://chromewebstore.google.com/detail/axi-bloqueur-doutil-ia/idbndcaecdbeegickfljfladfbogchok?hl=fr&utm_source=ext_sidebar

Pour Firefox :
Axi est aussi disponible depuis le store de Firefox ! Télécharge l'extension avec le lien ci-dessous :
Axi pour Firefox !

https://addons.mozilla.org/fr/firefox/addon/axi-bloqueur-d-outil-ia/

Téléchargez d'abord ce zip et dézippez le sur votre machine :
Le fichier ZIP de l'extension


Allez sur about:debugging#/setup, une fois ici, allez sur l'onglet "Ce firefox"

Contribuer à l'extension
Vous souhaitez contribuer et participer à Axi ! Voici notre GitHub, tu pourras participer au développement et avoir accès à notre documentation !
